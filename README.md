# Project Documentation

## Overview
This project is designed to provide a secure and efficient cryptocurrency handling mechanism. It includes various functionalities for managing digital assets securely.

## Features
- **Secure Transactions**: All transactions are encrypted.
- **User-Friendly Interface**: The application is easy to use and navigate.
- **Multi-Currency Support**: Support for various cryptocurrencies.

## Setup Instructions
1. **Clone the Repository**  
   Run the following command to clone the repository:  
   ```bash
   git clone https://github.com/kostaprag-crypto/Test.git
   ```

2. **Install Dependencies**  
   Navigate to the project directory and install the required dependencies:  
   ```bash
   cd Test
   npm install
   ```

3. **Configuration**  
   Create a `.env` file in the root directory and add the necessary configuration variables.

4. **Run the Application**  
   Use the following command to start the application:  
   ```bash
   npm start
   ```
